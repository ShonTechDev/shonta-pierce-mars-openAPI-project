Art Explorer - Open API Project

Overview:
Art Explorer allows users to explore artwork and artists from the Art Institute of Chicago using their public API. The project demonstrates how to retrieve and display data from an external API using JavaScript, HTML, and CSS. The user is able to browse the following features pulled directly from the museum's database:

Features:
Dynamic Image Rendering using the museum's IIIF imaging services
-Recent Artwork Gallery
-Random Artwork Viewer
-Artist Spotlight

This project pulls data from two API endpoints, providing multiple datapoints provided by the Art Institute of Chicago:
-Artworks Endpoint
-Artists Endpoint
These endpoints provide artwork titles, artists, artwork images, and artist details.

How to run:
1. Download the files from the current GitHub repository.
2. Make sure all project folders remain together and included (HTML, CSS, JSS, and images).
3. Locate and run the index.html file in your browser.
4. Use the buttons on the page to fetch artwork or artists from the API.

Technologies Used
-HTML, CSS, JS, Art Institute of Chicago Open API
